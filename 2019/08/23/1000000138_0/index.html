除了去拍亲子照，出门吃大餐

小编再给大家支两招

七夕之夜，全家一起来场浪漫约会吧！


快乐时光最必不可少的元素当然是美食。大餐过后，全家一起动手做一份浪漫甜点，享受不被打扰的亲子时光，给七夕节画上一个圆满的句号~

所谓我能想到最浪漫的事，就是和你一起甜甜蜜蜜的吃啊吃啊吃 :)

小编给大家推荐的以下两个甜点制作活动都是以水果为主，既美味、又健康。制作非常简单，宝宝也能一起参与，爸妈们别忘了用镜头记录宝宝动手的精彩瞬间。分享在朋友圈，晒晒不一样的七夕~


爱心一串串


全家一起来剥橘子，然后将橘子瓣切出半个“心”，再用牙签一串～

1秒变出心形果盘

第二步，在心形“城墙”的里和外分别填满不同的水果，比如以下的草莓和蓝莓：

然后拿掉“城墙”，当~当~当~当~

一个完美的心形就出现啦：
