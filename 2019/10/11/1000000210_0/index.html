最近国庆档热门电影《我和我的祖国》刷屏，
用我闺蜜的话形容就“让亿万中国人哭成狗”。
得知飘在美帝的我到目前为止
还没有看过这部电影，
小编被深深的鄙视了…
 
上网一搜，发现这部电影在北美地区
竟然是和国内同步上映的，
而且评价普遍都还不错：

不错，但如果我是中国人意义应该会更加深刻

先要说明的是，我不是中国人，也不是学中国历史的学生。在观看的过程中，我感觉这部电影应该会让经历过这些事的人很有共鸣，但是对于局外人来说，情感上的共鸣会比较小，这是不可避免的，因此在观看时请记住这一点。


话虽如此，我还是很喜欢。它非常注重细节，制作精良。有人告诉我，很多代表当时情况的细节（衣服，电视等）都非常准确。因此，我们怀旧的同时也可以了解到更多关于中国的历史。


我有点困惑的是因为它将不同的历史片段和故事融合在一起，所以很难知道它的真实性。（例如，第一个故事是关于一个真人真事，而第二个故事我就不知道是不是。）此外，七个故事确实在角色发展上有些仓促，可以理解但是确实比我期望看到的要仓促一些。


总的来说，我觉得从历史的角度还是值得一看。也许它会让你有兴趣来进一步了解中国历史。


此外，我还找到了一个我认为很棒的中国儿童演员，里面有个故事类似于惊奇队长，但做得更好。

非常精彩

我认为这是一件了不起的作品，所有居住在中国的人都知道那些年头非常不容易。中国人有光明的未来，来吧，继续前进。

有趣，有经过深思熟虑的影片

一部出色的电影，讲的一个个不同人生主题都是无名英雄故事。每个故事都是对中国重要历史时期的人们的生活的一瞥，而不是特别关注情节的发展。爱国主义的成分偏多，但是想想这部电影的目的也是预料之中的。总的来说，我觉得有两个故事非常出色，但全部七个作品都很有趣且鲜明，所以不会觉得很拖沓 。

和平&爱

我喜欢这部电影。我给它评9星。电影本身8分，个人经历1分。这部电影和我想得完全不一样， 我以为更多的是关注国家的，但事实并非如此。就像电影的名字一样，人民是先于国家的。我比较喜欢的地方是这部电影从个体的角度描述每个故事，反映中国发展过程中的重大事件是如何影响其中的个人的。你不必成为军人或科学家也一样可以做出贡献， 作为一个普通人也可以做很多事情。这部电影展示了朋友，家人，伴侣之间的爱，以及对土地，国家和人民的各种不同主题的爱。

以上评论摘自美国主流影评网imdb.com。
 
不过看影评是一方面，
从这些影评中，我们也可以来一起学习
一组非常实用的英语词汇，
那就是如何夸奖别人：

good [ɡʊd]

好的; 不错的; 符合标准的;令人满意的

例句：It is a good movie.

这部电影还不错。

great [ɡreɪt]

很好; 非常棒; (强调尺寸、体积或质量等) 很多很大的

例句：This is a great movie!

这部电影非常棒！

well done [wel ˈdʌn]

表现出色的; 熟透的

例句：This is a very well done film.

这是一部非常出色的电影。

wonderful [ˈwʌndərfl]

精彩的; 绝妙的; 令人惊奇的; 令人赞叹的;

例句：It was a wonderful movie!

这部电影太精彩了!

fantastic [fænˈtæstɪk]

极好的; 了不起的; 难以置信的;富于想象的;

例句：This is a fantastic work.

这是非常了不起的作品。

发现了没？这些词汇在
夸赞的语气程度上也是有所不同的~
比如wonderful和fantastic就要比
good或great的夸奖程度要高上好几个等级。

爸妈们也别忘了用英语来夸奖孩子一句：
Fantastic!
