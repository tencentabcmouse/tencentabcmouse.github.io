如何做

材料：

9之左右的冰棒棍、橡皮筋若干、胶水

瓶盖（或其它小容器），以及最后用于投射的小物件

建议使用棉球、兵乓球等安全不会伤到他人的小物


以下是小编制作的40秒全过程

大家看一遍就会做啦：


 

亲子观察、实验与讨论


1. 一开始，可以让孩子轻轻压下弹射端，弹出棉球

然后逐渐加强下压的力度

和孩子一起预测、观察棉球弹出的高度和距离有什么变化

 

2. 变换弹射支架（即十字交叉点）距离瓶盖的位置

和孩子一起预测、观察棉球弹出的高度和距离又有什么变化？

 

3. 改变弹射物体，比如将棉球换成兵乓球

比较不同物体的弹射结果，讨论可能的原因

还可以多弹射几次

看看那个物体弹出的距离范围相较比较稳定？

 

4. 小编在一位美国妈妈 Chelsey 的博客上

找到了一种更有趣的实验延伸

直接用一次性勺子代替瓶盖

不但制作更简单，还可以让孩子调节勺子伸出的长短

观察不同力臂长度带来的不同变化！

（图片来源：buggyandbuddy.com）

 


实验背后的科学道理

有兴趣的爸妈还可以在实验观察后

进一步和孩子讲讲其中的科学原理

 

投射器背后隐含的基本原理是：

能量可以在不同形式间转换

而且能从一个物体传递到另一个物体中

 

当我们下压弹射棒时，就产生了存储在木棒中的弹力

而我们放开的一瞬间，弹力就转化成了动力

并从木棒转移到了被弹射的棉球中

 

当然，以上只是最基础的解释

学霸爸妈还可以借这个小实验和孩子讲讲力矩、重力

的概念和作用
