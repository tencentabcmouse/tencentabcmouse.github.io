问

从最基础的问题开始：什么是自然拼读法？

Kimberly Oliver Burnim: 

简单地说，

自然拼读法就是字母的字形

与它们在单词中所代表的发音之间的对应关系。


举个例子，

当我们在一个单词中看到字母 b，

我们知道它通常发 /b/ 音（类似于“播”的发音）

就是我们在 boy 和 bicycle 这些单词中听到的 b 的发音

这就是自然拼读。


问

自然拼读法对孩子的英语学习有何帮助？

Kimberly Oliver Burnim: 

学习自然拼读法有助于孩子

准确、迅速地进行阅读和拼写单词。

孩子学会了自然拼读，当他们遇到生词时，

就能够运用拼读知识迅速 “解码” 生词。

 

问

什么叫解码呢？

Kimberly Oliver Burnim: 

解码是指，

当我们看到某个书面单词，

就能运用自然拼读规则（字母的字形所对应的发音）

准确地拼读出这个单词。


例如，如果孩子看到由 m-a-p 三个字母组成的单词，

而且知道m、a 和 p 这三个字母所代表的发音，

他们就能够通过这三个已知的字母发音拼读出单词 map 的发音。

小编总结：

结合以上的专家解读，
小编感觉其实英语中的自然拼读
就和我们语文中的“拼音”是一个道理：

只要学会了拼音，自然就能读出生词~


问

那如何帮助孩子学习自然拼读呢？

Kimberly Oliver Burnim: 

实际上，第一步仅仅就是

学习字母表中各个字母的名称。

研究发现，

当孩子们知道字母的名称时，

能够更好地掌握它们所代表的发音。

小编这里补充说明一下：

字母的名称和字母的发音是不同的概念~

比如字母D
它本身的名称读作/di/ (好像“地”的发音)
可它在单词中的发音通常是/d/ (好像“得”的发音) 


掌握了字母的名称，

第二步就是帮孩子学习每个字母的发音

——例如在和孩子一起阅读英语绘本、故事书的时候，

可以反复的指出单词中的某个字母，

并和孩子讨论这个字母在单词中的读音。


这里需要注意的是，

我们只需要从每个字母最常见的发音开始，

而不用急着教会孩子每个字母可能代表的所有发音。

举个例子：

字母 c 和字母 k 有时候都代表 /k/ 音，

比如在单词 cat 中；

而有时则与字母 s 代表相同发音 /s/，

例如在单词 city 中。

然而，字母 c 在大部分单词中的常见发音是 /k/ ，

因此，我们会首先教孩子掌握这个常见发音。
